##  Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mahendra2611/wonderla-task.git
   cd wonderla-task
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
